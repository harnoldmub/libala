Sujet : Développement du module d'invitation et génération PDF

Objectif : Automatiser la création et l'envoi d'invitations personnalisées depuis le backoffice.

Fonctionnalités requises :



Template : Création d'une invitation au format PDF (2 pages), basée sur la charte graphique du modèle fourni en pièce jointe.

QR Code Intelligent : Intégration d'un QR code unique sur l'invitation. Ce code devra être lié à l'ID de l'utilisateur pour permettre au staff de retrouver sa table lors du scan à l'entrée.

Automatisation :

L'administrateur clique sur "Envoyer l'invitation" depuis le profil d'un invité.

Le système génère le PDF instantanément avec les données de l'invité.

L'interface affiche un aperçu du PDF généré.

Un bouton "Télécharger" permet de récupérer le fichier localement.
